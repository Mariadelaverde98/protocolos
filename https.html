HTTPS es lo mismo que HTTP, con la salvedad de que se le añade el término “seguro”.

En el caso de que se utilice el protocolo HTTP, los datos serán accesibles para cualquier persona que pueda captar la comunicación. Sin embargo, el protocolo HTTPS utiliza una conexión segura ya que se basa en el sistema de cifrado SSL. Esto permite que los datos puedan viajar de manera segura de un dato a otro, y que no puedan ser captados por cualquiera.

¿Cómo funciona el protocolo HTTPS?

1. Abrimos una página de Internet y el navegador intentará conectarse a un sitio que ha sido protegido con SSL.

2. El navegador requerirá que el servidor web se identifique.

3. Será así como el servidor mandará una copia de su certificado SSL a nuestro navegador.

4. El navegador, una vez ha recibido la copia, comprobará si el sitio web es de confianza. Si es así, manda un mensaje que lo acredita.

5. El servidor mandará un acuse de recibo con firma digital que permitirá que se inicie la conexión cifrada.

6. Así es como empezarán a transferirse datos cifrados entre el servidor y el navegador.
